# linux-course
# Linux / DevOps Fundamentals Portfolio

## О проекте

В репозитории представлены практические работы, выполненные в рамках изучения Linux, сетевого взаимодействия, мониторинга и контейнеризации.  
Проекты охватывают полный цикл базовой подготовки: от установки и настройки системы до развёртывания сервисов и построения мониторинга.

---

# Выполненные направления

## 1. Операционные системы UNIX/Linux

В рамках данного блока реализованы базовые задачи администрирования Linux.

### Выполненные работы
- установка Ubuntu Server без графического интерфейса;
- настройка пользователей и прав доступа;
- настройка hostname, временной зоны и сетевых параметров;
- обновление системы и управление пакетами;
- настройка и использование SSH;
- работа с системными журналами;
- использование утилит мониторинга (`top`, `htop`, `df`, `du`, `fdisk`, `ncdu`);
- настройка и использование планировщика `cron`;
- работа с терминальными текстовыми редакторами.

---

## 2. Bash и автоматизация

Реализованы скрипты для автоматизации системных задач.

### Выполненные работы
- написание bash-скриптов с параметрами командной строки;
- проверка корректности входных данных;
- вывод системной информации;
- сохранение результатов работы в файл;
- реализация цветового оформления вывода (ANSI);
- использование конфигурационного файла для задания параметров;
- анализ файловой системы средствами shell;
- декомпозиция скриптов на модули.

---

## 3. Компьютерные сети в Linux

Выполнена настройка сетевого взаимодействия и исследование сетевых механизмов.

### Выполненные работы
- настройка IP-адресации и масок подсетей;
- конфигурация сетевых интерфейсов;
- реализация статической маршрутизации;
- анализ таблицы маршрутов;
- проверка соединения (`ping`, `traceroute`, `netstat`);
- измерение пропускной способности (`iperf3`);
- настройка firewall с использованием `iptables`;
- настройка NAT;
- работа с DHCP;
- настройка SSH-туннелей.

---

## 4. Мониторинг и анализ системы

Реализована система мониторинга и анализа состояния системы.

### Выполненные работы
- генерация файлов и заполнение файловой системы;
- реализация сценариев очистки системы;
- генерация логов в формате nginx;
- анализ логов;
- настройка GoAccess;
- настройка Prometheus для сбора метрик;
- настройка Grafana для визуализации;
- использование готовых дашбордов;
- реализация собственного `node_exporter`.

---

## 5. Docker и контейнеризация

Реализованы задачи по контейнеризации и развёртыванию сервисов.

### Выполненные работы
- загрузка и запуск Docker-образов;
- управление контейнерами;
- анализ параметров контейнера;
- работа с конфигурацией nginx внутри контейнера;
- изменение конфигурации и проверка работы сервиса;
- экспорт и импорт контейнеров;
- разработка мини веб-сервера (C + FastCGI);
- настройка nginx как reverse proxy;
- создание собственного Dockerfile;
- использование Dockle для анализа безопасности;
- настройка Docker Compose.

# Проекты

## Проект 1. Базовое администрирование Linux

Развёрнута и настроена серверная Linux-среда.

### Реализовано
- установка и базовая настройка ОС;
- конфигурация сети (DHCP и статические параметры);
- настройка пользователей и прав;
- работа с системными службами;
- анализ состояния системы и ресурсов;
- настройка автоматических задач (cron).

---

## Проект 2. Автоматизация на Bash

Реализованы сценарии автоматизации системных операций.

### Реализовано
- обработка аргументов командной строки;
- валидация входных данных;
- вывод и форматирование информации;
- работа с файлами и директориями;
- настройка конфигурации через внешний файл;
- визуализация вывода.

---

## Проект 3. Сетевое взаимодействие

Настроена сеть между виртуальными машинами.

### Реализовано
- конфигурация IP-адресов и маршрутов;
- настройка статической маршрутизации;
- проверка соединения между узлами;
- настройка firewall;
- анализ сетевого трафика;
- измерение скорости соединения;
- настройка NAT и DHCP.

---

## Проект 4. Мониторинг и логирование

Реализована система сбора и анализа данных.

### Реализовано
- генерация логов;
- анализ логов веб-сервера;
- визуализация логов (GoAccess);
- сбор метрик (Prometheus);
- построение дашбордов (Grafana);
- разработка собственного экспортера метрик.

---

## Проект 5. Контейнеризация и сервисы

Реализовано развёртывание сервисов в контейнерах.

### Реализовано
- запуск и управление контейнерами;
- настройка nginx;
- проксирование запросов;
- разработка веб-сервера;
- сборка Docker-образа;
- использование Docker Compose;
- анализ безопасности контейнеров.

---

# Итог

Репозиторий содержит практические реализации задач в следующих областях:
- администрирование Linux;
- автоматизация через Bash;
- настройка и диагностика сетей;
- мониторинг и анализ системы;
- контейнеризация и развёртывание сервисов.

Работы выполнены в среде Ubuntu Server 20.04 LTS с использованием виртуализации и CLI-инструментов.
